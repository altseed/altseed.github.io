Altseed

複数のプログラミング言語、複数の環境対応のゲームエンジン

主要な機能

対応環境

  • Windows
  • Mac(ツール未対応)
  • Linux(ツール未対応、ユーザーがAltseedをコンパイルする必要あり)

対応プログラミング言語

  • C++
  • C#

グラフィック

  • DirectX11、OpenGLによる2D描画(Windows)
  • OpenGLによる2D描画(Mac、Linux)
  • 画像描画
  • 図形描画
  • 文字描画
  • エフェクト描画

オーディオ

  • XAudio2によるオーディオ再生(Windows)
  • PulseAudioによるオーディオ再生(Linux)
  • OpenALによるオーディオ再生(Mac)
  • Wave、Ogg等のオーディオフォーマットに対応

入力

  • マウス入力
  • キーボード入力
  • ジョイスティック入力

ファイル

  • ファイルパッケージの入出力
